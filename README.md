# Loja LB Games

Este é um projeto de estudo de Front-End, criando uma interface para uma loja de jogos simulada chamada "LB Games". O projeto utiliza HTML, CSS (com Bootstrap) e JavaScript (incluindo jQuery).

## Estrutura do Projeto
- `index.html`: Página inicial com banners, promoções e produtos em destaque.
- `login.html`: Página para login e cadastro de usuários.
- `css/estilo.css`: Folha de estilos customizada.
- `js/script.js`: Scripts básicos (voltar ao topo, simulação de login).
- `js/jquery-script.js`: Scripts usando jQuery (animação do formulário de cadastro).
- `img/`: Diretório contendo as imagens utilizadas no layout (logo, banners, capas de jogos).

## Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- jQuery

## Como executar
- Basta clonar o repositório e abrir o arquivo `index.html` em qualquer navegador moderno.
